## About

This project helps developers prepare for coding interviews by teaching common DSA patterns. Instead of memorizing individual problems, you'll learn to recognize patterns that appear across multiple problems.

## Features

- **19 Core Patterns** covering 327+ LeetCode problems
- **Pattern-based organization** (Two Pointers, Sliding Window, DFS, BFS, etc.)
- **Interactive UI** with dark/light theme toggle
- **Problem sets** with hints and solutions
- **Theory explanations** for each pattern
- **Code examples** in JavaScript and C++
- **Responsive design** - works on mobile, tablet, and desktop
- **Search functionality** to find patterns quickly

## Tech Stack

- **React 19** - Latest React with modern hooks
- **Vite 6.0** - Fast build tool and dev server
- **Tailwind CSS 3** - Utility-first styling
- **React Router 7** - Client-side routing
- **React Icons** - Icon library
- Deployed on **GitHub Pages**

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation & Development

```bash
# Clone the repository
git clone https://github.com/ani8727/DSA_Learing_Hub.git
cd DSA_Learing_Hub

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Deploy to GitHub Pages
npm run deploy
```

Visit `http://localhost:5173` to see the app running locally.

## Patterns Covered

### Array Patterns
- **Two Pointers** - Efficient array/string traversal
- **Sliding Window** - Contiguous subarray problems
- **Fast & Slow Pointers** - Cycle detection, middle element
- **Merge Intervals** - Overlapping intervals
- **Cyclic Sort** - Missing/duplicate numbers

### Graph Patterns  
- **DFS** (Depth-First Search) - Graph traversal, backtracking
- **BFS** (Breadth-First Search) - Shortest path, level-order
- **Union-Find** - Connected components, cycle detection

### Tree Patterns
- **Tree DFS** - Preorder, inorder, postorder traversals
- **Tree BFS** - Level-order traversal
