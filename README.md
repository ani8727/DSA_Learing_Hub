## 📖 Project Overview

DSA Learning Hub is a comprehensive platform designed to help developers master Data Structures and Algorithms through a **pattern-based approach**. Built with modern technologies and featuring a sophisticated design system.

### ✨ Key Features

- 🎨 **Sophisticated Design System** - Professional color hierarchy with glass morphism effects
- 🌙 **Dark/Light Mode** - Seamless theme switching with optimized color schemes
- 📱 **Fully Responsive** - Perfect experience across all devices
- 🔄 **Pattern-Based Learning** - Focus on algorithmic patterns rather than individual problems
- 📊 **Interactive Components** - Engaging UI with smooth animations
- 🎯 **8+ Core Topics** - Arrays, Trees, Dynamic Programming, and more
- 💡 **100+ Curated Problems** - Hand-picked from easy to advanced levels

## 🎯 How to Add New Topics

Adding new topics is super easy! Just follow these steps:

### 1. Create New Page Component

Create `src/pages/YourTopic.jsx`:

```jsx
import React from 'react';
import Section from '../components/Section';
import CodeBlock from '../components/CodeBlock';

const YourTopic = () => {
  return (
    <div className="space-y-6">
      <div className="glass-effect rounded-xl p-8">
        <h1 className="text-4xl font-bold gradient-text mb-3">
          Your Topic 🚀
        </h1>
        <p className="text-gray-600 text-lg">Description here</p>
      </div>

      <Section title="Section Title" defaultOpen>
        <p>Your content here</p>
        <CodeBlock code={`// Your code here`} />
      </Section>
    </div>
  );
};

export default YourTopic;
```

### 2. Add Route in App.jsx

```jsx
import YourTopic from './pages/YourTopic';

// Add in Routes:
<Route path="/your-topic" element={<YourTopic />} />
```

### 3. Add to Sidebar

In `src/components/Sidebar.jsx`, add to topics array:

```jsx
{ path: '/your-topic', icon: YourIcon, label: 'Your Topic' }
```

That's it! 🎉
