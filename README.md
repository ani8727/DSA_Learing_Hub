## üì¶ Install & Run (Local)

Clone, install and run locally:

```bash
git clone https://github.com/ani8727/DSA_Learing_Hub.git
cd DSA_Learing_Hub
npm install
npm run dev
```

Build for production:

```bash
npm run build
# Preview the production build locally
npm run preview
```

## üåç Deployment Instructions

### GitHub Pages Deployment

The Portfolio is deployed on GitHub Pages at: [ani8727.github.io/DSA_Learing_Hub](#)

#### Deploy Steps:

1. Build the project:

   ```bash
   npm run build
   ```

   The `dist` folder is ready for deployment

2. Push to main branch:

   ```bash
   git add .
   git commit -m "Update Project"
   git push origin main
   ```

3. Deploy to GitHub Pages:

   ```bash
   npm run deploy
   ```

### Alternative Deployment Options

- **Vercel**: Auto-deploy from GitHub (recommended)
  - Connect repo to Vercel dashboard
  - Build command: `npm run build`
  - Output: `dist`

- **Netlify**: Static hosting
  - Build command: `npm run build`
  - Publish directory: `dist`
