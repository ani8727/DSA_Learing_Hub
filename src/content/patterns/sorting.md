# Sorting Patterns

## Overview
- Sorting arranges data in a particular order (ascending/descending).
- Types: Comparison-based (Bubble, Selection, Insertion, Merge, Quick, Heap), Non-comparison (Counting, Radix, Bucket)

## Tricks & Patterns
- Custom Comparator Functions
- Two Pointer with Sorting
- Counting Sort for small ranges
- Bucket/Index Mapping
- Stable vs Unstable Sort
